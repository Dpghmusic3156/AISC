# Plan 1.1: Project Setup & System Tray Foundation

## Objective
Initialize the base C# WPF project with .NET 8, and implement a System Tray icon so the application can run implicitly in the background.

## Context
- .gsd/SPEC.md
- .gsd/DECISIONS.md
- Use `H.NotifyIcon.Wpf` (or equivalent modern package) for Tray Icon.
- MaterialDesignThemes should be installed.
